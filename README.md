# 脳科学ゼミナール 事前学習サイト

京都女子高等学校・東山高等学校 共催「脳科学ゼミナール」の事前学習用サイト（GitHub Pages想定・ビルド不要）。
統合仕様は `SPEC.md` を参照（2つの制作プロンプトを1つに統合した「完璧なサイトの条件」）。

## フォルダ構成
```
site/
├── index.html        アプリの入口（1枚のみ）
├── SPEC.md            統合仕様書（完璧なサイトの条件）
├── css/
│   └── style.css       デザイントークン・全スタイル（青・白基調＋水色/紫アクセント）
├── js/
│   ├── data.js          教授情報・専門用語辞典(39語)・ゼミナール概要・比較表・チャット応答ルール
│   └── app.js            ルーティング／検索／メモ／チャット／SVGイラスト等のロジック
└── assets/              （現在未使用。実写真等を追加する場合はここへ）
```

## ページ一覧
ホーム／脳科学ゼミナールとは（新設）／教授紹介（吉沅洪教授が主役、見学教授を併記）／トラウマとは／脳の仕組み／心理療法（CBT・EMDR・精神分析の比較表つき）／専門用語辞典／参考文献／AIチャット／メモ

## 主な変更点（このバージョン）
- カラーパレットを「青・白基調＋水色・紫アクセント」に統一
- 「脳科学ゼミナールとは」ページを新設（目的・経緯・連携意義・スケジュール・参加メリット）
- 教授紹介を吉沅洪教授中心の詳細プロフィール（経歴・研究アプローチ・代表的な研究・研究動機）に再構成
- 心理療法ページにCBT／EMDR／精神分析の比較表を追加
- 専門用語を39語に拡充（神経伝達物質・CBTを追加）
- クイズ機能は不採用
- 教授の顔写真は肖像権の関係でイニシャルのプレースホルダー表示。実写真は `assets/` に配置し `SVG.avatarPlaceholder` 呼び出しを `<img>` に差し替えて対応

## 動かし方
`index.html` をブラウザで直接開くか、フォルダごとGitHub Pagesにアップロードしてください（ビルド不要）。
