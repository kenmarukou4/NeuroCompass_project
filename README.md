# 脳科学ゼミナール 事前学習サイト

東山高等学校の生徒が個人制作した「脳科学ゼミナール」事前学習用サイト（GitHub Pages想定・ビルド不要）。
統合仕様は `SPEC.md` を参照。

## フォルダ構成
```
site/
├── index.html        アプリの入口
├── SPEC.md            統合仕様書
├── css/style.css       デザイントークン・全スタイル（脳マップのアニメーション含む）
├── js/
│   ├── data.js          専門用語辞典(536語)・教授情報・ゼミナール概要・比較表
│   └── app.js            ルーティング／検索／メモ／チャット／脳マップ／画像表示ロジック
└── assets/terms/        用語ごとのPNG画像を追加できます
    ├── README.md          画像追加のルール説明
    └── term-image-list.csv  全536語の「ファイル名・用語名・カテゴリ」一覧
```

## 用語辞典への画像追加について
`assets/terms/{用語id}.png` という名前で画像を置くと、辞典カードと詳細ページに自動で表示されます。
どの用語にどのファイル名を使うかは `assets/terms/term-image-list.csv` を開いて確認してください。
画像がない用語はそのまま非表示になるだけなので、一部の用語だけ画像を用意する形でも問題ありません。

## 用語数についての進捗
現在536語まで拡充しました（目標1000語）。脳科学（解剖・生理・発達・分子・臨床神経学など）と
臨床心理（診断・心理療法・発達理論・社会心理・アセスメントなど）の両分野を幅広くカバーしていますが、
1000語にはまだ届いていません。品質を保ちながら、今後も継続してバッチで追加していきます。
