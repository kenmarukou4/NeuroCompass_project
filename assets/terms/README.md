# 専門用語の画像追加について

このフォルダに、各用語のPNG画像を追加すると、自動的に辞典カード・詳細ページに表示されます。

## 命名ルール
`assets/terms/{用語のid}.png`

例：
- ニューロン → `assets/terms/neuron.png`
- 扁桃体　　 → `assets/terms/amygdala.png`
- PTSD　　　→ `assets/terms/ptsd.png`

## どの用語がどのファイル名か
`term-image-list.csv` に、全用語の「ファイル名・用語名・カテゴリ」の一覧を出力しています。
Excel／スプレッドシートで開いて確認しながら画像を用意してください。

## 注意点
- 画像を追加していない用語は、何も表示されません（エラーにはなりません）。すべての用語に用意する必要はなく、分かりやすくなりそうな用語だけで大丈夫です。
- 画像サイズの目安：横長（例：800×500px程度）。カード表示・詳細表示ともに自動でトリミングされます。
- ファイル名は全て半角英数字・ハイフンです（日本語ファイル名にはしないでください）。
